# SumTimes 
## An addition and multiplication quiz

This mini project of mine is a simple, customizable quiz for those
who love to practice their mental math skills in both addition and
multiplication. 

The program is just a kind of **prototype** for the actual thing I am
planning to achieve. Due to the upcoming academic year, I have to 
postpone the second and better version of this project.

I really hope to implement this program with Flask, HTML, CSS and JavaScript when my semester break comes along.
